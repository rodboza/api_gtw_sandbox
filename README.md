# API Gateway SandBox (Master)

Para executar você precisa de uma máquina Linux com o Docker instalado.

No terminal digite:

``` sh
sudo su
git clone https://github.com/rodboza/api_gtw_sandbox.git
git checkout master
cd api_gtw_sandbox

```
